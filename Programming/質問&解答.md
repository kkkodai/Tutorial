- 1から入力した自然数まで掛ける

*私*

```python
def FirstFactorial(num): 

    # code goes here 
    temp = 1
    for N in range(1,num+1):
        temp = temp*N
    num = temp
    return num
    
# keep this function call here  
print FirstFactorial(raw_input())
```

```python
def FirstFactorial(num): 
    if num == 1:
        return 1
    else:
        return num * FirstFactorial(num - 1)
print FirstFactorial(raw_input())
```

- 最大の英文字列を抽出

*私*：9点
```python
import re
def LongestWord(sen): 

    # code goes here 
    match = re.split("[^a-zA-Z]",sen)
    temp = ""
    for m in match:
        if len(temp) < len(m):
            temp=m
    return temp
    
# keep this function call here  
print LongestWord(raw_input())
```

*プロ*
```
def LongestWord(sen): 
    sen = sen.translate(None, "!@#$%^&*()_+=-0987654321[]\;'./,<>?:\"{}|").split()
    big = ''
    for i in sen:
        if len(i) > len(big):
            big = i
    return big
print LongestWord(raw_input())
```
